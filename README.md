# Skills

个人工具库，包含以下实用技能：

---

## 🔍 codebase-inspection
分析代码库行数、语言构成、代码/注释比例。适合在想知道"这个项目有多大"、"各语言占比多少"时使用。

---

## 🛡️ command-life-guard
终端命令急救工具 — 当命令失败、超时、报 BLOCKED、权限错误、连接失败时使用。自动诊断原因并给出修复建议。

---

## ✏️ prompt-refiner
Prompt 结构化改进工具 — 当 AI 理解偏差、输出不理想时，诊断六大维度（角色/任务/约束/示例/格式/上下文）并输出改进版本。

---

## 🚀 first-commit-guide
Git 第一次提交指南 — 从零到第一次 push，让 git 不再神秘。解决"刚学编程不知道代码怎么管理"、"git init 之后不知道该干嘛"的痛点。

---

## 📊 trending-briefing
每日极客情报站 — 自动抓取 GitHub Trending，生成结构化日报。解决"没时间刷 GitHub 但想知道今天在流行什么"的痛点。
